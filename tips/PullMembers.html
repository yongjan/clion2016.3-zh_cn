The Pull Members Up refactoring allows you to move class members to a base class. The Push Members Down refactoring helps clean up the class hierarchy by moving class members to a subclass.

CLion highlights dependant members so you never miss one while refactoring your code.

To pull members up through the hierarchy press &shortcut:MemberPullUp; (Refactor | Pull Members Up...). To push members down press &shortcut:MemberPushDown; (Refactor | Push Members Down...).
